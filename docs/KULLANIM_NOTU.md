# Kullanım Notu

## PC programını açma

```text
tools/MASAUSTU_PROGRAMI_BASLAT.bat
```

Açılmazsa önce:

```text
tools/PYTHON_MODUL_KUR.bat
```

çalıştır.

## Windows açılınca otomatik başlasın

```text
tools/BASLANGICA_EKLE.bat
```

çalıştır.

## Kalibrasyon

1. PC programı açık olacak.
2. Pico üzerindeki TEST / KALİBRASYON butonuna basılacak.
3. Tam ekran kalibrasyon açılacak.
4. Hedef köşeye silah çevrilecek.
5. BOMBA / SEÇ butonuna basılacak.
6. 4 köşe bitince otomatik kaydedip kapanacak.

## Titreşimi azaltma

Programda `Titreşimi Azalt` bölümünden sürgüyü sağa al.

- Az: Daha hızlı tepki, daha az filtre.
- Çok: Daha az titreme, biraz daha yumuşak tepki.

## Önemli

Kalibrasyonda imleci butona götürmek yoktur. Köşe kaydı fiziksel BOMBA / SEÇ butonuyla yapılır.
